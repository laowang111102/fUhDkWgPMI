# 校园用车管理系统 python1185

## 项目概述

校园用车管理系统是一款集成学生信息管理、校车信息管理、教师信息管理、司机信息管理、校车记录管理、学生乘坐管理以及教师乘坐管理等功能的综合管理系统。通过全方位管理校园内车辆使用，提高车辆使用效率，确保运行安全性。

## 技术栈

- 后端：Django 2.0、pymysql、click、django-threadlocals、requests、xlrd 1.2.0、django-cors-headers
- 前端：Vue 2
- 运行环境：Python 3.7.7、MySQL 5.7、Node 12、cnpm 7.1.0

## 功能模块

### 学生信息管理

- 管理学生基本信息：姓名、学号、班级等
- 确保信息准确性与完整性

### 校车信息管理

- 记录与管理校车基本信息：车牌号、车型、座位数等
- 提供校车调度与分配基础数据支持

### 教师信息管理

- 管理教师个人信息：姓名、教工号、所属部门等
- 提供校车乘坐与教师考勤数据支持

### 司机信息管理

- 管理校车司机个人信息：姓名、驾驶证号、驾驶经验等
- 确保校车司机资质与安全性

### 校车记录管理

- 记录校车运行记录：发车时间、到达时间、行驶路线等
- 提供数据支持，监控校车实时运行情况

### 学生乘坐管理

- 管理学生校车乘坐信息：乘坐时间、乘坐路线等
- 确保学生乘坐准确性与安全性

### 教师乘坐管理

- 管理教师校车乘坐信息：乘坐时间、乘坐路线等
- 提供便捷的校车乘坐服务

## 系统角色

系统主要涉及以下角色：

- 学生
- 教师
- 司机
- 管理员
